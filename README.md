## README
### Life of Nicole

This is a simple multi-level game created using the Javascript game engine Phaser.

#### Build Instructions

1. Clone repository
2. [Start a web server](http://phaser.io/tutorials/getting-started/part2)
3. Navigate to your web server and start playing!
